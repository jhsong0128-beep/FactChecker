# ✅ 참고 출처 섹션 대폭 업그레이드 완료!

## 🎉 새로운 기능들

### 1. 📂 세션별 분류
출처가 **3가지 카테고리**로 깔끔하게 정리됩니다:

- **📄 학술논문** - 학술지, 연구 논문
- **📚 도서** - 단행본, 교재
- **🌐 인터넷 자료** - 웹사이트, 온라인 기사

각 카테고리마다 **총 건수**가 표시됩니다!

### 2. 🖼️ 미리보기 카드
각 자료가 카드 형태로 표시되며:

- 📄 아이콘/썸네일
- 📖 제목
- 👤 저자/출판사/연도
- 📝 간단한 요약
- 📚 대출 가능 여부 배지

### 3. 📚 도서관 정보
**대출 가능** 자료에는:

- ✅ "대출 가능" 초록색 배지
- 📍 도서관 위치 목록
  - 국립중앙도서관
  - 서울대학교 중앙도서관
  - 서울시립도서관 등

### 4. 🔽 더보기 기능
자료가 많을 때:

- 처음엔 **2개만** 표시
- **"더보기"** 버튼 클릭하면 전체 표시
- 남은 자료 수도 함께 표시 (예: "더보기 (4건 더) ▼")
- **"접기"** 버튼으로 다시 줄일 수 있음

### 5. 💡 상세 모달창
자료 카드를 **클릭**하면:

```
┌─────────────────────────────┐
│  자료 상세 정보              │
├─────────────────────────────┤
│  • 저자, 출판사, 발행연도   │
│  • ISBN/DOI                  │
│  • 상세 요약                 │
│                              │
│  📚 도서관 정보              │
│  ✓ 대출 가능                │
│                              │
│  열람/대출 가능한 도서관:    │
│  📍 국립중앙도서관           │
│  📍 서울대학교 중앙도서관    │
│  📍 서울시립도서관           │
│                              │
│  [원문 보기] [도서관 예약]   │
└─────────────────────────────┘
```

## 💻 기술 구현

### 데이터 구조
```typescript
interface Source {
  id: number
  title: string
  author?: string
  publisher?: string
  year?: string
  type: 'academic' | 'book' | 'web'
  thumbnail: string
  summary: string
  library?: {
    available: boolean
    locations: string[]
  }
}
```

### 상태 관리
- `showMoreAcademic`: 학술논문 더보기 상태
- `showMoreBooks`: 도서 더보기 상태
- `showMoreWeb`: 인터넷 자료 더보기 상태
- `selectedSource`: 선택된 자료 (모달용)
