- [ ] 3단계 신뢰도 평가 로직

**테스트 예시**:
```javascript
describe('AI Verification', () => {
  it('should return "trusted" for well-known facts', async () => {
    const result = await verify('지구는 태양 주위를 돈다');
    expect(result.trustLevel).toBe('trusted');
  });
  
  it('should return "suspicious" for unverified claims', async () => {
    const result = await verify('알려지지 않은 허위 주장');
    expect(result.trustLevel).toBe('suspicious');
  });
});
```
