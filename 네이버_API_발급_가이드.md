# 📚 네이버 API 발급 가이드 (학술자료 + 인터넷 자료)

## 🎯 네이버 API 장점:
- ✅ **완전 무료**
- ✅ **즉시 발급** (승인 대기 없음!)
- ✅ **하루 25,000건** 무료
- ✅ **학술 검색 + 웹 검색** 한 번에!
- ✅ **한글 검색 완벽 지원**

---

## 📋 발급 방법 (5분)

### 1단계: 네이버 개발자 센터 접속

```
https://developers.naver.com/
```

### 2단계: 로그인 및 애플리케이션 등록

1. 네이버 계정으로 로그인
2. 우측 상단 **"Application"** → **"애플리케이션 등록"** 클릭

### 3단계: 애플리케이션 정보 입력

**애플리케이션 이름:**
```
FactChecker
```

**사용 API 선택:** (중요!)
- ☑️ **검색** ← 이것 체크!
  - 블로그 검색
  - 뉴스 검색
  - 책 검색
  - 백과사전 검색
  - 카페글 검색
  - 지식iN 검색
  - 학술 검색 (중요!)
  - 웹문서 검색 (중요!)

**비로그인 오픈 API 서비스 환경:**
- Web 서비스 URL: `http://localhost:3000`

### 4단계: 등록 완료

등록하면 즉시 다음이 발급됩니다:
- **Client ID** (예: `abcd1234efgh5678`)
- **Client Secret** (예: `XyZ123abc`)
